# 🧠 General Knowledge Quiz Game

A simple and interactive **Python Quiz Game** that tests general knowledge skills. This project was developed as part of my **Decode Labs Internship**.

---

## 🚀 Project Overview

This is a command-line quiz game where users answer multiple-choice/general questions and receive a final score at the end.

It demonstrates basic Python concepts like:

* Conditional statements
* User input handling
* Variables
* Score tracking
* String comparison

---

## ✨ Features

✅ Interactive quiz format
✅ Immediate feedback for each answer
✅ Score tracking system
✅ Simple and beginner-friendly CLI game
✅ Final score display
